# shop_backend

## Compte de démonstration pour les recruteurs

## Objectif du projet

Ce projet n'a pas pour but de créer un design parfait ou une interface ultra-polishée.

L'objectif principal est de maîtriser les concepts de **e-commerce**, notamment :
- Gestion des produits et catégories
- Gestion des utilisateurs et rôles (admin / client)
- Gestion des commandes et des statuts via **Stripe**
- Flux complet d'achat, de paiement et de suivi de commande

> L’accent est mis sur la **logique métier et la maîtrise des outils**, plutôt que sur l’apparence visuelle.
